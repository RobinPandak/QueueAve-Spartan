What changes

  • Card wrapper — white card with rounded corners + subtle shadow, floated on the warm background
  • Labels — uppercase, bold, small tracking (like badminton) instead of plain small text
  • Template hint — shows the metrics in the selected template (updates live on change)
  • Focus ring — coral focus ring + border on active field
  • Submit button — pill shape, coral shadow glow, loading spinner state
  • Back link — "← [Event name]" at top for clear navigation context
  • Convert to client component — needed for loading state + template hint reactivity

What stays the same

  • All 4 fields: template, date, group, notes — same fields, same server action
  • Date defaults to today
  • Group defaults to "All groups"
  • Redirects to the session results grid after creation
