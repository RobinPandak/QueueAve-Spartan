What changes vs. current wizard

  • 4 steps instead of 3 — split name into its own focused Step 1 (like badminton's basics step)
  • Progress bars — thin segment bars at top (one per step), coral fill for completed
  • Step label — "Step X of 4, [step name]" below bars
  • Large heading per step — conversational question format ("What's your event called?")
  • Name suggestions chips — clickable chips on Step 1
  • Slide animation — steps slide in/out (SlideIn component, same as badminton)
  • Uppercase bold labels with letter-spacing on all fields
  • Coral focus ring + coral Continue button (replacing step circle indicators)

What stays the same

  • All existing fields — name, date, venue, description, groups, templates, metrics
  • Dynamic add/remove for groups and templates
  • Same createEvent server action, same redirect on success
