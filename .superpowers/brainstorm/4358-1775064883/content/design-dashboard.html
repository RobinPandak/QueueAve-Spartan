What's included

  • Greeting — "Good morning/afternoon/evening, [coach name]" from organizers.name
  • Stats row — active count (coral), total events, total participants
  • Search — shown once coach has 5+ events
  • Filter tabs — All / Active / Draft / Past (client-side filtering, no server round-trip)
  • Active section — pulsing teal dot, 2-col grid (3-col on desktop), coral left-border accent cards, share link button
  • Past section — muted cards for completed + draft events
  • Share join link — copies or uses Web Share API (same pattern as badminton)
  • Empty state — keep existing Spartan empty state, just restyle the button
